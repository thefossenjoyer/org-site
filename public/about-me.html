My OS

My current daily driver is GuixOS, a GNU/Linux distro based on the guix package manager. Guix is a functional pkg manager, highly inspired by Nix.

As it shouldn't be suprising, by default there isn't any non-free software in guix's repos. Talking about non-free software, Guix System uses Linux libre instead of the full Linux kernel. The reason for that is quite obvious. There are some proprietary blobs in the kernel (drivers..).

Of course, the community found a way around these limitations. There is a repo with non-free software. Also, System Crafters provides a custom ISO with the full Linux kernel and the repo already set up.
